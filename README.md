# gary0413/homebrew-tap

Homebrew tap for [SayKey](https://github.com/gary0413/SayKey) — local
push-to-talk voice input for macOS, built for Chinese–English code-switching.

```bash
brew install --cask gary0413/tap/saykey
```

This installs the app, pulls in `whisper-cpp` + `opencc`, and downloads the
whisper + VAD models to `~/.saykey/models` on first install.
